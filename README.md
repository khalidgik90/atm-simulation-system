# ATM Simulation System

A Python-based ATM simulation that allows users to create accounts, deposit/withdraw money, check balances, and view admin analytics.

## Features
- Secure account creation with 4-digit PIN
- Deposit and withdrawal operations
- Balance checking
- Admin analytics panel
- Data persistence using CSV (Pandas)
- Bank statistics using NumPy
- Balance visualization with Matplotlib

## Built With
- Python
- Pandas
- NumPy
- Matplotlib

## How to Run
```
python project.py
```
